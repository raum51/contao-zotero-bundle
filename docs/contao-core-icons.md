# Contao Core – Verfügbare Backend-Icons

**Stand:** Februar 2026  
**Quelle:** [contao/core-bundle](https://github.com/contao/contao/tree/5.x/core-bundle/contao/themes/flexible/icons) (Flexible Theme)  
**Zweck:** Referenz aller im Contao-Core enthaltenen Icons für DCA-Operationen und Backend-Anpassungen.

---

## Hinweise

- **Pfad in DCA:** Icons werden per Dateinamen referenziert (z. B. `'icon' => 'visible.svg'`).
- **Varianten:** Viele Icons existieren zusätzlich als `--dark.svg` (Dark Mode), `--disabled.svg` und Kombinationen. Contao wählt die passende Variante automatisch.
- **Vorschau-URL:** `https://raw.githubusercontent.com/contao/contao/5.x/core-bundle/contao/themes/flexible/icons/{DATEINAME}`
